3) What is the use of the map() function in JavaScript? How is it different from forEach()?
    Ans: map() Function একটি array এর প্রতিটি উপাদানের উপর কাজ করে , নতুন একটি array রিটার্ন করে, And মূল array পরিবর্তন করে না।
         forEach() Function array এর প্রতিটি উপাদানের উপর কাজ করে, কিছু রিটার্ন করে না (undefined রিটার্ন করে) And সাধারণত লুপ চালানোর জন্য ব্যবহার হয়
    
5) What is the difference between == and ===?
   Ans: == শুধু মান (value) তুলনা করে, And প্রয়োজনে টাইপ পরিবর্তন (type conversion) করে।
    === মান এবং টাইপ দুইটাই তুলনা করে, And টাইপ পরিবর্তন করে না।

9) Explain the concept of Scope in JavaScript (Global, Function, Block).
    Ans: Scope মানে হলো — কোনো ভ্যারিয়েবল কোথা থেকে অ্যাক্সেস করা যাবে।
   Global Scope : 1. গ্লোবালি ডিক্লেয়ার করা ভ্যারিয়েবল. 2. পুরো প্রোগ্রাম থেকে অ্যাক্সেস করা যায়।
   Function Scope: 1. function এর ভিতরে ডিক্লেয়ার করা ভ্যারিয়েবল. 2. শুধুমাত্র সেই function এর ভিতরে ব্যবহার করা যায়।
   Block Scope: 1. { } এর ভিতরে ডিক্লেয়ার করা ভ্যারিয়েবল. 2. শুধু সেই ব্লকের ভিতরে ব্যবহার করা যায়।
